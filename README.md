# Jason Ranney - Southeast Hard Surface Territory Manager Website

A modern React website showcasing Jason Ranney's services as a Southeast Hard Surface Territory Manager for Shaw Industries, specializing in Coretec Shaw Floors including Shaw Laminate, Shaw Vinyl, and Shaw Hardwood.

## Features

- **Responsive Design**: Mobile-friendly layout that works on all devices
- **Modern UI**: Clean, professional design with smooth animations
- **Interactive Forms**: Sample request and contact forms with validation
- **Brand Showcase**: Highlighting premium Shaw flooring products
- **Navigation**: Smooth single-page application navigation

## Pages

1. **Home**: Hero section with call-to-action buttons and brand showcase
2. **About**: Information about Jason Ranney and his services
3. **Order Samples**: Form to request flooring samples
4. **Contact**: Contact form and direct contact information

## Technologies Used

- React 18
- CSS3 with modern features (Grid, Flexbox, CSS Variables)
- Responsive design principles
- Modern JavaScript (ES6+)

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm (comes with Node.js)

### Installation

1. Clone or download this project
2. Navigate to the project directory
3. Install dependencies:
   ```bash
   npm install
   ```

### Running the Development Server

```bash
npm start
```

This will start the development server and open the website in your default browser at `http://localhost:3000`.

### Building for Production

```bash
npm run build
```

This creates an optimized production build in the `build` folder.

## Customization

### Colors
The website uses a blue color scheme that can be customized in `src/App.css`:
- Primary: `#1e3a8a`
- Secondary: `#3b82f6`
- Accent: `#10b981`

### Content
Update the content in each component file to match your specific needs:
- Company information
- Contact details
- Product descriptions
- Images and logos

### Styling
Modify `src/App.css` to change:
- Colors and typography
- Layout and spacing
- Animations and transitions
- Responsive breakpoints
